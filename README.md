## Projetos em Destaque

### ADV7ia — Stack Local de IA
`IC01-aops/ADV7ia/`

Mesh de controle para tarefas de IA local em Arch Linux. Integra LM Studio (modelo de linguagem local), OpenHands, Aider e Cline. Implementa rollover de estado de tarefas e proxy LAN via Caddy. Inclui suporte a VM via túnel reverso (GNOME Boxes).

### JustVibing/Snippets — BugBounty Recon Toolkit
`JustVibing/Snippets/`

Toolkit client-side para reconhecimento em bug bounty. Versão 1.0.3 com security score 95/100. Analisa scripts externos, variáveis globais e formulários ocultos. Correções de segurança aplicadas: CORS, race condition em download, vazamento de memória, validação RFC-791, proteção contra ReDoS.

### AVAL01-IC — Projeto Técnico de Segurança em IoT
`IC01-aops/AVAL01-IC/`

Projeto técnico da disciplina Introdução à Computação (UniCEUB). Análise de segurança do dispositivo MXQ Pro 4k (firmware Android comprometido), com proposta de arquitetura defensiva baseada em OpenBox, IDS/IPS (ADV7Sec) e segmentação de rede. Fontes primárias: FBI PSA, HUMAN Security, Dr.Web, Anatel.
