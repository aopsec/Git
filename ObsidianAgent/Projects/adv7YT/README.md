# adv7ProTools

---

*Dedico este programa ao meu pai — por tudo.*

---

## O que é o adv7ProTools?

O **adv7ProTools** é um programa para Windows que faz duas coisas:

1. **Baixar vídeos e músicas** da internet — do YouTube, TikTok, Instagram, SoundCloud, Vimeo e mais de 1.700 outros sites — direto para o seu computador.
2. **Converter arquivos de vídeo e áudio** que você já tem no computador para outro formato.

Tudo isso sem precisar instalar mais nada. O programa já vem com tudo que precisa dentro dele.

---

## Requisitos mínimos

- Windows 10 ou Windows 11 (64 bits)
- Conexão com a internet (apenas para baixar vídeos)
- Espaço em disco suficiente para os arquivos que você vai baixar ou converter

---

## Como instalar (passo a passo)

### Passo 1 — Baixar o programa

1. Faça o download do arquivo `adv7ProTools0.0.1v.exe`.
2. Salve em qualquer pasta do seu computador — por exemplo, na Área de Trabalho ou em `Documentos`.

### Passo 2 — Abrir o programa

1. Localize o arquivo `adv7ProTools0.0.1v.exe` onde você salvou.
2. Dê **dois cliques** nele para abrir.

> **Pode aparecer um aviso do Windows dizendo "Windows protegeu seu computador".**
> Isso acontece porque o programa não tem uma assinatura digital paga. Ele é seguro.
> Para continuar: clique em **"Mais informações"** e depois em **"Executar assim mesmo"**.

3. O programa vai abrir direto — sem instalação, sem próximo/próximo/concluir.

## Funcionalidade 1 — Baixar vídeos e músicas

### O que você pode baixar

O programa funciona com mais de **1.700 sites**, incluindo:

| Site | Exemplos do que dá para baixar |
|---|---|
| **YouTube** | Vídeos, clipes, shorts, aulas, documentários |
| **TikTok** | Vídeos do feed |
| **Instagram** | Reels, vídeos de posts |
| **Vimeo** | Vídeos de criadores |
| **SoundCloud** | Músicas e podcasts |
| E muito mais | Facebook, Twitter/X, Reddit, Twitch e outros |

### Passo a passo para baixar um vídeo

**Passo 1 — Copiar o link do vídeo**

1. Abra o site (por exemplo, YouTube) no seu navegador.
2. Entre no vídeo que você quer baixar.
3. Clique na barra de endereço do navegador (onde aparece o link, no topo da tela).
4. Selecione tudo (`Ctrl + A`) e copie (`Ctrl + C`).

**Passo 2 — Colar o link no programa**

1. Na aba **⬇ Download**, clique no botão **Paste** (Colar).
   - O link que você copiou vai aparecer automaticamente no campo.
   - Você também pode clicar no campo e pressionar `Ctrl + V` para colar manualmente.

**Passo 3 — Escolher a qualidade**

Selecione uma das três opções de qualidade:

| Opção | O que significa | Quando usar |
|---|---|---|
| **Native** | Melhor qualidade disponível no site | Quando você quer a melhor imagem possível |
| **FHD 1080p** | Full HD, máximo de 1080 linhas | Boa qualidade sem arquivo muito grande |
| **Fast** | Qualidade reduzida, arquivo pequeno | Quando o espaço em disco ou a internet é limitada |

> **Dica:** Se você não sabe qual escolher, deixe em **Native**.

**Passo 4 — Escolher o formato do arquivo**

| Formato | O que é | Quando usar |
|---|---|---|
| **MP4** | Formato mais comum de vídeo | Para assistir no celular, TV ou computador |
| **MKV** | Formato que preserva mais detalhes | Para arquivar com a melhor qualidade |
| **WebM** | Formato para a web | Para usar em sites |

> **Dica:** Na dúvida, escolha **MP4**. Funciona em qualquer lugar.

**Passo 5 — Escolher onde salvar**

1. Olhe o campo **Pasta de saída** na parte de baixo da tela.
2. Se você quiser salvar em outro lugar, clique em **Browse** e escolha a pasta.
3. Por padrão, os arquivos são salvos na pasta **Vídeos** do seu usuário.

**Passo 6 — Iniciar o download**

1. Clique no botão **Download** (no canto inferior direito da aba).
2. A barra de progresso vai começar a se mover.
3. O status mostra a velocidade e o tempo restante.
4. Quando terminar, o status mostra **"Done → [nome da pasta]"**.

**Para cancelar antes de terminar:** clique no botão **Cancel**.

## Funcionalidade 2 — Converter arquivos

### O que é converter?

Converter significa transformar um arquivo de vídeo ou áudio que você já tem no computador em outro formato. Por exemplo:

- Transformar um `.mp4` em `.mp3` para ouvir só o áudio no celular.
- Transformar um `.avi` antigo em `.mp4` para assistir na TV nova.
- Transformar um vídeo em um `.gif` animado para mandar no WhatsApp.

### Passo a passo para converter um arquivo

**Passo 1 — Ir para a aba de conversão**

Clique na aba **🔄 Converter** no topo do programa.

**Passo 2 — Selecionar o arquivo**

1. Clique no botão **Browse…** ao lado do campo "Select a file to convert…".
2. Uma janela vai abrir para você navegar nas pastas.
3. Encontre o arquivo que quer converter, clique nele e depois clique em **Abrir**.
4. O caminho do arquivo vai aparecer no campo.

**Passo 3 — Escolher o formato de saída**

Clique na lista suspensa (a caixa que mostra os formatos). Uma lista com **30 opções** vai aparecer, dividida em 4 grupos:

#### Grupo Vídeo (10 formatos)

| Formato | Extensão | Para que serve |
|---|---|---|
| **MP4** | `.mp4` | O mais universal. Funciona em qualquer aparelho ou programa |
| **MKV** | `.mkv` | Ótimo para arquivar. Suporta legendas e várias faixas de áudio |
| **AVI** | `.avi` | Formato antigo. Use só se o programa de destino exigir |
| **WebM** | `.webm` | Otimizado para a internet e players de navegador |
| **MOV** | `.mov` | Formato da Apple. Comum em macOS e iPhones |
| **TS** | `.ts` | Transmissão de TV e gravações de OBS |
| **HEVC MP4** | `.mp4` | H.265. Arquivo ~50% menor que MP4 normal com a mesma qualidade |
| **FLV** | `.flv` | Flash Video. Formato antigo para plataformas legadas |
| **WMV** | `.wmv` | Windows Media Video. Para compatibilidade com programas antigos |
| **3GP** | `.3gp` | Formato para celulares antigos. Arquivos bem pequenos |

#### Grupo Áudio (10 formatos)

| Formato | Extensão | Para que serve |
|---|---|---|
| **MP3** | `.mp3` | O formato de música mais conhecido. Funciona em tudo |
| **AAC** | `.aac` | Melhor qualidade que MP3 no mesmo tamanho |
| **WAV** | `.wav` | Áudio sem compressão. Máxima qualidade, arquivo grande |
| **FLAC** | `.flac` | Compressão sem perda. Qualidade de CD com metade do tamanho do WAV |
| **OGG** | `.ogg` | Formato aberto. Bom para músicas e podcasts |
| **M4A** | `.m4a` | Formato padrão da Apple Music e iTunes |
| **WMA** | `.wma` | Windows Media Audio. Para compatibilidade com Windows antigo |
| **OPUS** | `.opus` | Excelente qualidade em arquivos pequenos. Ideal para voz |
| **AIFF** | `.aiff` | Formato da Apple sem compressão. Igual ao WAV, mas para Mac |
| **AC3** | `.ac3` | Dolby Digital Surround. Padrão de Blu-ray e DVD |

#### Grupo Imagem (4 formatos)

| Formato | Extensão | Para que serve |
|---|---|---|
| **Animated GIF** | `.gif` | GIF animado a 10fps, 480px. Para mandar no WhatsApp ou usar na web |
| **Animated WebP** | `.webp` | Animação moderna, ~40% menor que GIF com mais cores |
| **PNG Frames** | `.png` | Salva cada segundo do vídeo como uma imagem PNG separada |
| **JPEG Frames** | `.jpg` | Salva cada segundo do vídeo como uma imagem JPEG separada |

> **Atenção para PNG Frames e JPEG Frames:** ao converter, o programa cria uma **pasta** com o nome do vídeo e salva uma imagem por segundo dentro dela. Por exemplo: `meu_video/frame_0001.png`, `meu_video/frame_0002.png`, etc.

#### Grupo Projeto/Edição Profissional (6 formatos)

Estes formatos são para quem usa programas de edição de vídeo profissional:

| Formato | Para qual programa |
|---|---|
| **ProRes .mov** | Final Cut Pro, DaVinci Resolve |
| **ProRes 4444 .mov** | After Effects (com canal alfa/transparência) |
| **DNxHD .mxf** | Avid Media Composer |
| **DNxHR HQ .mxf** | Avid Media Composer (versões mais recentes) |
| **H.264 .mov** | Adobe Premiere, DaVinci Resolve |
| **CineForm .mov** | GoPro Studio, Premiere |

> **Dica:** Passe o mouse em cima de qualquer formato na lista para ver uma descrição detalhada de 8 segundos.

**Passo 4 — Conferir a pasta de saída**

O arquivo convertido vai ser salvo na mesma pasta que aparece em **Pasta de saída**. Se quiser mudar, clique em **Browse**.

> **Proteção automática:** Se você converter um arquivo para o mesmo formato e na mesma pasta, o programa **não vai sobrescrever** o original. Ele vai salvar com o nome `arquivo_converted.mp4`, por exemplo.

**Passo 5 — Iniciar a conversão**

1. Clique no botão **Convert** (canto inferior direito da aba).
2. A barra de progresso vai se mover durante a conversão.
3. Quando terminar, o status mostra **"Done → [caminho do arquivo]"**.

**Para cancelar antes de terminar:** clique no botão **Cancel**.

## Funcionalidades extras

### Histórico

O programa guarda uma lista de tudo que você baixou ou converteu.

- Clique em **▶ Histórico** para expandir a lista.
- Cada entrada mostra: data/hora, tipo (Download ou Convert), e se foi bem-sucedido.
- Para limpar o histórico, expanda-o e clique em **Clear**.

### Log (Registro de atividade)

- Clique em **▶ Log** para ver em tempo real o que o programa está fazendo por baixo dos panos.
- Útil para entender o que está acontecendo caso algo dê errado.

### Tema claro / escuro

- No canto superior direito, há um interruptor com **Light** e **Dark**.
- Clique nele para alternar entre o tema claro e o tema escuro.

## Perguntas frequentes

**O programa precisa de internet para funcionar?**
Só para baixar vídeos. Para converter arquivos que já estão no computador, não precisa de internet.

**Onde os arquivos baixados são salvos?**
Por padrão, na pasta **Vídeos** do seu usuário (`C:\Users\SeuNome\Videos`). Você pode mudar isso clicando em **Browse** ao lado de **Pasta de saída**.

**Por que o programa demora para começar a baixar?**
Nos primeiros segundos, ele está identificando o vídeo e escolhendo a melhor qualidade disponível. Isso é normal.

**Posso baixar só o áudio de um vídeo do YouTube?**
Sim. Baixe normalmente e depois use a aba **Converter** para transformar o arquivo `.mp4` em `.mp3` ou `.aac`.

**O que acontece se eu fechar o programa no meio de um download?**
O download é interrompido e o arquivo parcial é descartado.

**O programa aparece como vírus no antivírus. É seguro?**
Sim. Programas pequenos sem assinatura digital paga são frequentemente marcados por engano pelos antivírus. O programa não acessa internet exceto para fazer os downloads que você pede, e não acessa nenhum arquivo que você não indicar.

**A barra de progresso fica parada no início. É normal?**
Nos primeiros segundos do download, o programa está processando as informações do vídeo. A barra começa a se mover assim que o download de fato começa.
