### Step 2: Create Windows-Compatible MCP Wrapper Scripts

The existing scripts use Linux paths (`/home/aops/OPia/Git/...`). Create Windows-adapted versions:

**mcp-fs-mesh-windows.sh**:
```bash
#!/usr/bin/env bash
set -euo pipefail
ROOT="$HOME/.local/mcp-servers/filesystem"
TARGET="${MESH_FS_TARGET:-/mnt/c/Users/AOPSec/Desktop/Git}"
BIN="$ROOT/node_modules/.bin/mcp-server-filesystem"
[ -x "$BIN" ] || { echo "[ERROR] filesystem MCP not installed: $BIN" >&2; exit 127; }
exec "$BIN" "$TARGET"
```

**mcp-git-mesh-windows.sh**:
```bash
#!/usr/bin/env bash
set -euo pipefail
PY="python3"
REPO="${MESH_GIT_REPO:-/mnt/c/Users/AOPSec/Desktop/Git}"
[ -d "$REPO/.git" ] || { echo "[ERROR] not a git repo: $REPO" >&2; exit 2; }
exec "$PY" -m mcp_server_git --repository "$REPO" "$@"
```

**mcp-vault-mesh-windows.sh**:
```bash
#!/usr/bin/env bash
set -euo pipefail
ROOT="$HOME/.local/mcp-servers/filesystem"
TARGET="${MESH_VAULT_TARGET:-/mnt/c/Users/AOPSec/Desktop/Git/ObsidianAgent/Vault}"
BIN="$ROOT/node_modules/.bin/mcp-server-filesystem"
[ -x "$BIN" ] || { echo "[ERROR] filesystem MCP not installed: $BIN" >&2; exit 127; }
exec "$BIN" "$TARGET"
```

**mcp-exec-mesh-windows.sh**:
```bash
#!/usr/bin/env bash
set -euo pipefail
HERE="$(cd "$(dirname "$0")" && pwd)"
PY="$HERE/.venv/bin/python"
[ -x "$PY" ] || { echo "[ERROR] executor venv missing: $PY" >&2; exit 127; }
exec "$PY" "$HERE/exec_mesh.py"
```
